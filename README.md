# Colorant

## Intro

- Sets text colors and effects in console.

## Features

- Foreground and background colors
  - Black
  - Red
  - Green
  - Yellow
  - Blue
  - Magenta
  - Cyan
  - White
  - Bright black
  - Bright red
  - Bright green
  - Bright yellow
  - Bright blue
  - Bright magenta
  - Bright cyan
  - Bright white
- Decorations
  - Bold
  - Underline
  - Reversed
